2. 2050.相交线条分割平面
要使平面尽可能多，第n条线要与前面n-1条线都有交点，且不发生三线交与一点
增加n个交点，多了n+1条线段，每个线段把一个平面分成两个平面，则增加n+1个平面。
F(1) = 2, F(n) = F(n-1) + n

3. 
两条平行线：第n次增加4n-2，总面数2n平方+1

折线：每多一条折线即两条平行线变一条交线，减少一个面，总面数2n平方-n+1 
	或者理解成第n条折线与前面n-1条折线增加22(n-1)个交点，即4(n-1)+1条线段，则增加个平面
F(n) = F(n-1) + 4(n-1) + 1

5. n个男女生排队站 女生不能单独站
第n个是男生：前n-1合法，则合法，前n-1非法，则非法，故为F(n-1)
第n个是女生：
n-2合法+女，则合法F(n-2).     
n-2非法 即 n-4合法+男+女，则合法F(n-4)
F(n) = F(n-1) + F(n-2) + F(n-4)

F(n) = F(n-1) + 2 * F(n-2)

7. （卡特兰数列）1 2 5 14 42 132...    2n个数字顺时针围成一圈，n条线两两相连，不能交叉

12，则有子问题345678，即F(3)
14，则有两个独立子问题23和5678，即F(1) * F(2)

F(n) = 2 * (F(0) * F(n-1) + F(1) * F(n-2) + F(2) * F(n-3) + ...... + F(n-1) * F(0))

```
int main(){
    int n, m, i, j, count;
    int scoreArr[50][5];
    int sbAll[50];//学生总分
    int majAll[5];//课程总分
    while(scanf("%d%d", & n, &m) != EOF){
        memset(sbAll, 0, sizeof(int));
        memset(majAll, 0, sizeof(int));
        for (i = 0; i < n; i++) {
            for (j = 0; j < m; j++) {
                scanf("%d", &scoreArr[i][j]);
                sbAll[i] += scoreArr[i][j];
                majAll[j] += scoreArr[i][j];
            }
        }
        for (i = 0; i < n; i++) {
            printf("%.2lf%c", sbAll[i] * 1.0 / m, i < n - 1 ? ' ' : '\n');
        }
        for (i = 0; i < m; i++) {
            printf("%.2lf%c", majAll[i] * 1.0 / n, i < m - 1 ? ' ' : '\n');
        }
        count = n;
        for (i = 0; i < n; i++) {
            for (j = 0; j < m; j++) {
                if(scoreArr[i][j] < majAll[j] * 1.0 / n){
                    count--;//一科不满足就去掉
                    break;
                }
            }
        }
        printf("%d\n\n", count);
    }
    return 0;
}
```
